# 📊 Power BI License Optimization & Usage Analytics

## 🧾 Project Overview
This project focuses on analyzing enterprise-wide usage of Microsoft Power BI licenses to identify underutilization, optimize license allocation, and reduce overall costs. The solution enabled data-driven decision-making for license governance and achieved significant cost savings.

---

## 🎯 Objective
- Analyze usage patterns of Power BI licenses across the organization
- Identify unused, underutilized, and misallocated licenses
- Align user roles with appropriate license types
- Reduce unnecessary licensing costs

---

## 📦 Scope & Approach
- Analyzed data for **6,986 total users**
- Categorized users into:
  - Consumer+
  - Creator
  - Viewer
  - Inactive
- Mapped user behavior against license requirements:
  - Download / Share
  - Report Creation
  - Dashboard Usage
- Built an interactive dashboard tracking:
  - Daily Active Users (DAU)
  - Peak usage time
  - Report execution volume
  - Export activity
  - Concurrent usage

---

## 🔍 Key Insights
- **License Underutilization**
  - 127 inactive Pro users
  - 420 wasted Pro licenses
  - 6 wasted PPU licenses

- **License Misallocation**
  - Many Pro users are only consuming reports (Viewer-level usage)
  - 73 users require Pro but do not have it

- **User Activity Distribution**
  - ~77% users are Viewers
  - Small percentage are active Creators

- **Low Engagement**
  - Avg DAU: 59 users
  - Weekly active users: 479

- **Usage Segmentation**
  - Heavy Users: 125
  - Medium Users: 64
  - Light Users: 299

---

## 💡 Optimization Strategy
- **Downgrade Licenses**
  - Convert inactive and viewer-only users to Free/Viewer licenses

- **Reallocate Licenses**
  - Assign Pro licenses to active creators

- **Remove Redundant Licenses**
  - Eliminate unused Pro and PPU licenses

- **Role-Based Licensing**
  - Creators → Pro / PPU  
  - Consumers → Free / Viewer
